# Gradio

Gradio é uma biblioteca que trabalha com **interfaces gráficas**. Ela não é tão popular, mas é a mais simples das mostradas no curso.

A ideia dessa aula é conhecer um pouco essa biblioteca, assim como o propósito dela seria para **construir e compartilhar aplicações focadas em machine learning**. Não vamos trabalhar com machine learning aqui, o objetivo nessa seção é construir aplicações gerais do Python.

## Para instalar
```bash 
pip install gradio
```

## Execução:
Para executar, digite no terminal:   
`python nome_do_arquivo.py`

A interface vai abrir em uma URL como:  
`http://127.0.0.1/7860`  
Copie ela e cole no navegador para acessar a interface.


## Relembrando que...
O Gradio não para substituir frameworks web como Django, Flask e etc. O foco do Gradio e de outras bibliotecas é hospedar e disponibilizar aplicações mais rápidas, focadas em machine learning. Quando a gente tá trabalhando com uma aplicação de machine learning ou um data app, a gente não tá tão interessado na interface, no layout, no design, estamos interessados em ter uma interface de usuário para interagir com o nosso modelo preditivo. Ela tem uma interface mais simples, mas isso não significa que você não possa customizar a interface.
