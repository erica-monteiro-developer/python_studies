# Gradio

Gradio é uma biblioteca que trabalha com **interfaces gráficas**. Ela não é tão popular, mas é a mais simples.

A ideia dessa aula é conhecer um pouco essa biblioteca, assim como o propósito dela seria para **construir e compartilhar aplicações focadas em machine learning**. Não vamos trabalhar com machine learning aqui, o objetivo nessa seção é construir aplicações gerais do Python.
